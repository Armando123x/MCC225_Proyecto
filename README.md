## Proyecto: Predicción de Spread-F mediante un enfoque multimodal

El fenomeno Spread-F es un evento diario ionosferico que afecta a la navegación por GPS y comunicaciones por radiofrecuencia a larga distancia.

![Ionosfera](https://cdn.shopify.com/s/files/1/0172/1591/4084/files/trimble-ionoguard-technology-ionosphere_600x600.png)

En el presente repositorio se adjunta el trabajo de enfoque multimodal para predicción de Spread-F: una modalidad incorpora datos de potencia medidos con el radar de Jicamarca y la otra modalidad incorpora datos geofisicos (K index, DST index, F10.7 index, IMF index, others).

Ambas modalidades compartirán información a fin de determinar si en la noche existirá Spread-F.
